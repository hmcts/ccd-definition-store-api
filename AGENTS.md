# Agent Instructions

This repository keeps workflow instructions under `docs/skills/`.

## Security

For JWT issuer validation, Spring Security, IDAM/OIDC, Helm/Jenkins issuer settings, and related regression testing, follow `docs/skills/security/SKILL.md`.

## Extending

Add new repo-local workflows under `docs/skills/<topic>/SKILL.md` and register them here so they are easy to discover and invoke consistently.
